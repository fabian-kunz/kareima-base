# Feature Modul Template

Dieses Template ist die Startstruktur für neue Funktionsbereiche.

## Namenskonvention

- Ersetze "Example" durch den Domain-Namen, z. B. "Machine" oder "Offer".
- Dateinamen und Typnamen konsistent umbenennen.

## Ziel

- Einheitliche Struktur in allen künftigen Modulen
- Trennung von View, Store, Service und Types
- Schneller Start ohne Boilerplate-Suche
